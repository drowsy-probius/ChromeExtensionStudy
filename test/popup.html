    prepare for verilog HDL practice (file 'test.v' modified)

    We will briefly cover Verilog, an Hardware Description Language (HDL) in this class. Verilog is an language to describe hardware components, and can implement combinational and sequential logic with high-level language syntax. In order to implement a hardware block through Verilog HDL, you will need a verilog compiler (as we need C compliler to use C language) 1. How to install A free verilog compiler is available at http://iverilog.icarus.com/ For those of you likes download a single executable file to install the verilog compiler, go to http://bleyer.org/icarus/ which has executable installer for windows. Download and run installer, then you will see that a directory appears: (assuming you installed to C:\ directory) c:\iverilog iverilog compiler runs in command prompt.  2. What tools to use You basically need three tools: iverilog: to compile a verilog file, generates netlist output vvp: converts the compiled output to a vcd (value change dump) file gtkview: graphical simulation tool to view the waveform from vcd file. 3. Example I will upload a sample verilog file called test.v (verilog files have extension .v) Download it to your computer and execute the following in the command prompt: 1) c:\iverilog\bin\iverilog -o test_output test.v 2) c:\iverilog\bin\vvp test_output 3) c:\iverilog\gtkwave\bin\gtkwave test_output.vcd You will see a simulation output which shows the waveform.
